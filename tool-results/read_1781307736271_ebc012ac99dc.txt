    22→## Key Adaptations (PostgreSQL → SQLite)

    26→- Views → Prisma queries with equivalent logic
    27→- RLS → API-level filtering by user role

   293→2. `/src/app/api/reportes/afp/download/route.ts` — AFP SEPP report CSV download

   295→
   296→### Implementation Details
   297→
   298→#### ISSS OIS Download (`/api/reportes/isss/download?mes=&anio=`)
   299→- **Auth**: Bearer token required, roles: ADMIN, GERENCIA, AUDITOR (401/403 on failure)
   300→- **Data Source**: Queries `parametroLegal` for ISSS rates + `empleado` for active employees with ISSS numbers + `planilla` for actual calculated amounts
   301→- **CSV Format**: Semicolon-delimited with BOM (`\uFEFF`) for Excel compatibility
   302→- **Header rows**: Periodo, Fecha de Generación, Tasa Laboral (3%), Tasa Patronal (7.5%), Tope Cotización
   303→- **Data columns**: Número ISSS, DUI, Nombre del Trabajador, Salario Cotizable, Cotización Laboral, Cotización Patronal, Total Cotización
   304→- **Footer**: Totals row with employee count and sum amounts
   305→- **Filename**: `OIS-{MM}-{YYYY}.csv`
   306→
   307→#### AFP SEPP Download (`/api/reportes/afp/download?mes=&anio=`)
   308→- **Auth**: Same RBAC as ISSS
   309→- **Data Source**: Queries `parametroLegal` for AFP rates + `empleado` for active employees with AFP data + `planilla` for actual amounts
   310→- **CSV Format**: Same semicolon+BOM format
   311→- **Header rows**: Periodo, Fecha de Generación, Tasa Laboral (7.25%), Tasa Patronal (8.75%)
   312→- **Data columns**: NUP, DUI, Nombre del Trabajador, Administradora AFP, IBC, Cotización Laboral, Cotización Patronal, Total Cotización
   313→- **Grouping**: Employees grouped by AFP administradora (CRECER, CONFIA) with subtotals per group
   314→- **Footer**: Grand totals row
   315→- **Filename**: `SEPP-{MM}-{YYYY}.csv`
   316→
   317→#### ISR F-910 Download (`/api/reportes/isr/download?mes=&anio=`)
   318→- **Auth**: Same RBAC as ISSS
   319→- **Data Source**: Queries `parametroLegal` with `tramos_isr` for ISR calculation + `empleado` for all active employees + `planilla` for actual amounts
   320→- **CSV Format**: Same semicolon+BOM format
   321→- **Header rows**: Periodo, Fecha de Generación, ISSS/AFP rates, Tope ISSS, plus full ISR tramo table (Desde, Hasta, Porcentaje, Cuota Fija)
   322→- **Data columns**: DUI, Nombre del Trabajador, Salario Bruto, ISSS Laboral, AFP Laboral, Renta Neta, ISR Retenido, Tramo ISR
   323→- **ISR Calculation**: Full tramo-based calculation matching the payroll engine, with planilla detail override when available
   324→- **Footer**: Totals row
   325→- **Filename**: `F-910-{MM}-{YYYY}.csv`
   326→
   327→### Design Decisions
   328→- Used planilla detail data when available (CALCULADA/APROBADA/PAGADA states) for accurate actual amounts, falling back to on-the-fly calculation
   329→- Semicolon delimiter (Latin American standard for Excel)
   330→- BOM prefix for proper UTF-8 display in Excel
   331→- Numbers formatted with 2 decimal places and $ prefix
   332→- Dates in DD/MM/YYYY format
   333→- All three endpoints follow the same auth/RBAC pattern for consistency
