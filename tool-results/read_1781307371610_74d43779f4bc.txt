    15→    15→## Architecture Decisions
    16→    16→- Using Next.js 16 with App Router (required)
    17→    17→- Using Prisma with SQLite (environment constraint) - business logic from PG triggers/functions moved to application layer
    18→    18→- All 26 views rendered as client-side navigation within single / route
    19→    19→- JWT-based authentication with role-based access control
    20→    20→- Payroll calculation engine implemented in TypeScript (matching legal formulas from Manual de Cumplimiento)
    21→    21→
    22→    22→## Key Adaptations (PostgreSQL → SQLite)
    23→    23→- ENUMs → String types with validation
    24→    24→- UUID → cuid() auto-generated IDs  
    25→    25→- Triggers → Application-layer middleware/hooks
    26→    26→- Views → Prisma queries with equivalent logic
    27→    27→- RLS → API-level filtering by user role
    28→    28→- pgcrypto → bcrypt for passwords, crypto for OTP
    29→    29→

   295→   295→
   296→   296→### Implementation Details
   297→   297→
   298→   298→#### ISSS OIS Download (`/api/reportes/isss/download?mes=&anio=`)
   299→   299→- **Auth**: Bearer token required, roles: ADMIN, GERENCIA, AUDITOR (401/403 on failure)
   300→   300→- **Data Source**: Queries `parametroLegal` for ISSS rates + `empleado` for active employees with ISSS numbers + `planilla` for actual calculated amounts
   301→   301→- **CSV Format**: Semicolon-delimited with BOM (`\uFEFF`) for Excel compatibility
   302→   302→- **Header rows**: Periodo, Fecha de Generación, Tasa Laboral (3%), Tasa Patronal (7.5%), Tope Cotización
   303→   303→- **Data columns**: Número ISSS, DUI, Nombre del Trabajador, Salario Cotizable, Cotización Laboral, Cotización Patronal, Total Cotización
   304→   304→- **Footer**: Totals row with employee count and sum amounts
   305→   305→- **Filename**: `OIS-{MM}-{YYYY}.csv`
   306→   306→
   307→   307→#### AFP SEPP Download (`/api/reportes/afp/download?mes=&anio=`)
   308→   308→- **Auth**: Same RBAC as ISSS
   309→   309→- **Data Source**: Queries `parametroLegal` for AFP rates + `empleado` for active employees with AFP data + `planilla` for actual amounts
   310→   310→- **CSV Format**: Same semicolon+BOM format
   311→   311→- **Header rows**: Periodo, Fecha de Generación, Tasa Laboral (7.25%), Tasa Patronal (8.75%)
   312→   312→- **Data columns**: NUP, DUI, Nombre del Trabajador, Administradora AFP, IBC, Cotización Laboral, Cotización Patronal, Total Cotización
   313→   313→- **Grouping**: Employees grouped by AFP administradora (CRECER, CONFIA) with subtotals per group
   314→   314→- **Footer**: Grand totals row
   315→   315→- **Filename**: `SEPP-{MM}-{YYYY}.csv`
   316→   316→
   317→   317→#### ISR F-910 Download (`/api/reportes/isr/download?mes=&anio=`)
   318→   318→- **Auth**: Same RBAC as ISSS
   319→   319→- **Data Source**: Queries `parametroLegal` with `tramos_isr` for ISR calculation + `empleado` for all active employees + `planilla` for actual amounts
   320→   320→- **CSV Format**: Same semicolon+BOM format
   321→   321→- **Header rows**: Periodo, Fecha de Generación, ISSS/AFP rates, Tope ISSS, plus full ISR tramo table (Desde, Hasta, Porcentaje, Cuota Fija)
   322→   322→- **Data columns**: DUI, Nombre del Trabajador, Salario Bruto, ISSS Laboral, AFP Laboral, Renta Neta, ISR Retenido, Tramo ISR
   323→   323→- **ISR Calculation**: Full tramo-based calculation matching the payroll engine, with planilla detail override when available
   324→   324→- **Footer**: Totals row
   325→   325→- **Filename**: `F-910-{MM}-{YYYY}.csv`
   326→   326→
